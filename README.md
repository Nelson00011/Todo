## Description
**Todo List** is a **React.js** application that is meant provide a simple todo list function. 
- The items are saved in a ***Postgres*** database. 


## Technology Stack
**Front-End:** HTML5, CSS, React.js

**APIs:**   

**Back-End:**  Postgres, Node.js
